This was the first video game character I've ever created from start to end. It was done in the "Advanced 3D Modelling" subject at CITM, UPC. It was really fun for me to create such an iconic monster and one of my ever favourite sci-fi movie character.

In this project I went through the whole creative process, I made the High-Poly sculpt, the retopology, the UV Unwrap, texturing, rigging, animation and rendering. If you check the link to ArtStation you'll see this work with all detail.
